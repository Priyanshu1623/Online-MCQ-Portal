# 📝 Modern Online MCQ Quiz Portal

A highly interactive and modern Multiple Choice Question (MCQ) web application built using HTML, CSS, and Vanilla JavaScript. The portal tests your knowledge of fundamental Web Development concepts.

## 🚀 Live Demo

You can play the live quiz here: [https://priyanshu1623.github.io/Online-MCQ-Portal/](https://priyanshu1623.github.io/Online-MCQ-Portal/)

It features a premium **Glassmorphism UI**, smooth hover animations, and an animated gradient background for an engaging user experience.

## ✨ Features

- **20 Dynamic Questions**: Carefully curated questions on HTML, CSS, and JS.
- **Modern UI/UX**: Designed using Glassmorphism and CSS animations.
- **Interactive Feedback**: Highlights the selected option dynamically.
- **Score Tracking**: Calculates and displays the final score at the end of the quiz.
- **Fully Responsive**: Works seamlessly on both desktop and mobile screens.
- **No Dependencies**: Built completely using pure Vanilla JavaScript, CSS3, and HTML5 (No external libraries required).

## 🛠️ Tech Stack

- **HTML5**: For the structure of the web application.
- **CSS3**: For styling, animated background, and Glassmorphism effects.
- **JavaScript**: For DOM manipulation, question loading, and score calculation.
